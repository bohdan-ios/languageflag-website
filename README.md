# LanguageFlag

> Know your language at a glance — a tiny menu bar app that shows your active keyboard layout as a flag.

🇺🇸 🇺🇦 🇪🇸 — switch layouts, see the flag instantly.

**For a limited time: free.**

[Download](https://github.com/bohdan-ios/LanguageFlag/releases/latest) · [Install via Homebrew](#install-via-homebrew) · [Website](https://bohdan-ios.github.io/languageflag-website/)

## Install via Homebrew

```bash
brew install --cask bohdan-ios/languageflag/languageflag
```

## Requirements

- macOS 12 or later
